# Doggers

Aplicación móvil desarrollada con React Native enfocada en el bienestar animal y la visibilización de animales en situación de abandono, extravío o necesidad de ayuda.

## Objetivo

Crear una plataforma que permita a la comunidad reportar, visualizar y dar seguimiento a casos relacionados con animales que necesiten ayuda, fomentando la conciencia y la participación ciudadana.

## Estado Actual

### Implementado

* Mapa principal integrado
* Navegación entre pantallas
* Pantalla Home
* Pantalla Cuenta
* Pantalla Reports
* Pantalla Reubicación
* Proyecto configurado con Git y GitHub

### En desarrollo

* Sistema de usuarios
* Creación de reportes
* Gestión de reubicaciones
* Integración con base de datos

## Tecnologías

* React Native CLI
* TypeScript
* React Navigation

* Git / GitHub

## Requisitos de Desarrollo

* Node.js 20+
* NPM 10+
* JDK 17
* Android Studio
* Android SDK 33
* Emulador Pixel 4 (Android 13)

## Instalación

```bash
git clone <URL_DEL_REPOSITORIO>
cd Doggers
npm install
npx react-native run-android
```
